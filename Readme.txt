Site criado pelas alunas Fernanda Vicente e Isabele Lemos no curso técnico de desenvolvimento de sistemas da Escola Técnica Porto Digital.

O projeto se trata de uma landing page com informações sobre a escola. Para o desenvolvimento do projeto foram utilizadas as seguintes tecnologias: HTML5, CSS3, JavaScript e Bootstrap5. O desafio foi criar um site com funcionalidades e responsivo para diversos formatos de telas.
